Features

Everything you need to manage Claude sessions

A local dashboard purpose-built for running multiple Claude Code instances in parallel.

▶

Live Terminal

Full xterm.js terminal with real-time I/O for each session. Read output, type input, just like a native terminal.

●

Session States

Five distinct states with visual indicators. A pulsing yellow badge tells you instantly when Claude needs your input.

↻

Session Persistence

Sessions survive page refreshes and reconnects. Auto-resume picks up right where you left off.

⚋

Git + JIRA Integration

Automatic branch tracking and JIRA ticket detection. Clickable status links keep context at your fingertips.

⌨

Keyboard-First

Rebindable shortcuts for everything — cycle instances, jump to a session by number, launch new instances, all from the keyboard.

➥

Hook Integration

External Claude Code instances auto-appear in the dashboard via hook scripts. No manual setup required.

☰

Project Grouping

Instances auto-group by project directory when working across repos. Collapsible headers keep the sidebar organized.

⚙

Per-Project Config

Drop a .mob/config.json in any repo for setup scripts, teardown hooks, and default launch settings.

🔈

Notification Sounds

Optional audio ping when Claude needs input — works even when the tab is visible and focused.
